# Describe

Describe Kubemq cluster command

## Synopsis

Describe command allows describing a Kubemq cluster to console

```text
kubemqctl get cluster describe [flags]
```

## Examples

```text
    # Describe Kubemq cluster to console
    kubemqctl get cluster describe
```
