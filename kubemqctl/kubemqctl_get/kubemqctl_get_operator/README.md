# Operator

## Synopsis

Get Kubemq Operators List

```text
kubemqctl get operator [flags]
```

## Examples

```text
    # Get Kubemq operators list 
    kubemqctl get operators  
    # Get Kubemq operator pod logs 
    kubemqctl get operators log
```
