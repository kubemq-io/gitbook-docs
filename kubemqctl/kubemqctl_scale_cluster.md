## kubemqctl scale cluster

Scale Kubemq cluster replicas command

### Synopsis

Scale command allows to scale Kubemq cluster replicas

```
kubemqctl scale cluster [flags]
```

### Examples

```

	# Scale Kubemq cluster  
	kubemqctl scale cluster 5

```
