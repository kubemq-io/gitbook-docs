## kubemqctl delete operator

Delete Kubemq operator

### Synopsis

Delete one or more Kubemq operators

```
kubemqctl delete operator [flags]
```

### Examples

```

	# Delete Kubemq operator 
	kubemqctl delete operator  

```

### Options

```
  -h, --help         help for operator
      --remove-all   remove all operator components
```
